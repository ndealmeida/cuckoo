# cuckoo
Cuckoo is my Python playground. Here's the stack of choice:
- Package Manager: uv
- Linter: ruff
- Web Framework: FastAPI
- Web Server: Uvicorn
- DI Container: injector
- Config Management: Dynaconf
- Scheduler: Rocketry
- Database: asyncpg (Postgres)
- Cache: redis

## Running

```sh
uv run cuckoo
```
